Anh Bình cứ upload file `.md` đó lên nhé. Dựa vào nội dung file, mình sẽ dùng công cụ `skill-creator` để:
1. Thiết lập cấu trúc thư mục skill chuẩn.
2. Viết file `SKILL.md` với frontmatter (name, description) chính xác để OpenClaw nhận diện được.
3. Tổ chức lại nội dung (chia ra thư mục `references/` hoặc `scripts/` nếu cần) để tối ưu context window.

Anh gửi file lên, mình sẽ bắt đầu ngay ạ!
