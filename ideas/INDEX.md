## 🏷️ Trạng thái

- ⚪ **Draft** - Ý tưởng thô, chưa rõ ràng
- 🟡 **In Progress** - Đang phác thảo, bổ sung chi tiết
- ✅ **Ready to Build** - Đã đủ thông tin, sẵn sàng build
- ✅ **Completed** - Đã hoàn thành và triển khai

---

## 📝 Cách Sử Dụng

1. **Tạo file mới** cho ý tưởng:
   ```bash
   cp ideas/TEMPLATE.md ideas/ten-y-tuong.md
   ```

2. **Phát triển ý tưởng** trong file đó (thêm requirement, design, user story...)

3. **Cập nhật INDEX.md** với thông tin ý tưởng

4. **Khi sẵn sàng** → báo cáo và bắt đầu build sản phẩm

---

**Lưu ý:** Mỗi file ý tưởng nên có cấu trúc:
- Mục tiêu
- Yêu cầu chức năng
- Thiết kế (database, API, UI)
- Các bước triển khai
- Dependencies/tech stack
- Timeline estimate
